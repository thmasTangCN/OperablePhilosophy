# 操作哲学 · Operable Philosophy

> *身处这个"草台班子"世界；我们信也没用，不信也没用；物理宇宙才不管我们信什么，说什么；它只在我们作死时，轻轻地，摁死我们！——像摁灭一根碍眼的烟头一样。*

**《操作哲学》不是一个需要被膜拜的"真理体系"，而是一套基于公理、可被公共检验的生存工具集。**

它将哲学从"说服的艺术"转变为**"免罚的工程"**：不承诺天堂，只帮助避免地狱；不规定你该怎样活，只在你行动之前告诉你——**这么干，会不会死？**

---

## 🧰 这是什么？

这是一本**开源哲学书**。它的核心姿态是：**别信，去验。**

*   **公理化基底**：从"差异即存在"出发，通过"穷举-校验"筛选出的一套硬核框架。
*   **旗舰工具**：提供 `M-I协同流` 分析模型、`三轨分析法`、`哲学推理机`（科学算命法）、`意识原型机`等用于分析意识与世界互动的操作工具。
*   **AI原生**：本书在设计上原生支持与AI协同工作。将书上传给任意语言模型，即可让它作为你的"思想义肢"，帮你进行复杂的**哲学推演**或**意识的体外模拟**。

这不是一本供在书架上的经典，而是一把**锤子**。你用它砸开现实中的钉子，好用就留着，不好用就扔。

## 🔬 核心概念速览

*   **差异即存在**：无差异则无存在。存在的起点是可观测的差异。
*   **M-I协同流**：任何对象都是一段物质(M)与信息(I)耦合的动态"流"，不可分割。
*   **有限观察者**：我们只能截断宇宙的极小片段，无法全知，只能"赌"。
*   **生存元价值**：对于生命，生存不是可选项，而是定义项。物理宇宙的铡刀强迫我们"必须求存"。
*   **真理即免罚**：一个道理是否为真，不看它多美、多自洽，只看按它行动会不会被现实惩罚（死或痛）。
*   **三轨分析法**（旗舰工具）：将任何理念或计划投入三个轨道进行"生存压力测试"——轨道一（理念极致推演）、轨道二（社会损益评估）、轨道三（自然损益评估——物理宇宙的终极铡刀）。

## 🤝 如何参与

这本书不属于我一个人。它是一场**公共验证的实验**。

你可以通过以下方式参与和贡献：

*   **验证断言**：使用AI或自行查证，检验书中已有的断言（或你自己推导出的新断言）是否与现实相符。将你的验证过程和结果整理成文，提交到 `examples/` 目录下。
*   **提交"验尸报告"**：用本书框架分析一个现实中的失败案例（如商业崩溃、政策失灵、社会危机），展示三轨分析法的实际运用。
*   **撰写应用指南**：分享你在某个垂直领域（如教育、投资、家庭关系）使用本书工具的心得和操作手册。
*   **抓虫与修正**：发现任何逻辑断裂、事实错误或表述不清之处，欢迎提交Issue或PR。
*   **翻译与传播**：帮助将本书或其衍生工具翻译成其他语言。

我们拒绝"释经人"。所有的讨论都应聚焦于：**这个断言能通过物理宇宙的检验吗？**
